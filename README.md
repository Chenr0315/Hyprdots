### 主要

| 软件包                                                                           | 用途           |
| -------------------------------------------------------------------------------- | -------------- |
| hyprland                                                                         | 平铺式合成器   |
| waybar                                                                           | 任务栏         |
| kitty                                                                            | 终端模拟器     |
| fish                                                                             | 终端           |
| starship                                                                         | 终端美化       |
| zen-browser                                                                      | 浏览器         |
| wofi                                                                             | 应用启动器     |
| nemo                                                                             | 文件管理器     |
| nemo-fileroller                                                                  | 压缩文件管理器 |
| dunst                                                                            | 通知           |
| cliphist                                                                         | 剪贴板         |
| brightnessctl                                                                    | 亮度调整       |
| fastfetch                                                                        | 信息显示       |
| btop                                                                             | 任务管理器     |
| gamemode                                                                         | 游戏模式       |
| hyprshot                                                                         | 截屏           |
| wlogout                                                                          | 注销菜单       |
| maplemono-nf-cn                                                                  | 字体           |
| material-icon-theme-git                                                          | 图标主题       |
| [Tokyonight-GTK-Theme](https://github.com/Fausto-Korpsvart/Tokyonight-GTK-Theme) | GTK 主题       |
